### 为什么它让程序员和极客们如此兴奋？

简单来说，Moltbot 是一个运行在你本地电脑（Mac、Windows 或 Linux）上的**全能 Agent**。

以往我们用 Claude 或 ChatGPT，数据都存在大公司的服务器里，你得主动登录网页去找它。而 Moltbot 彻底打破了这个界限：

1. **它住在你的聊天软件里：** 你不需要打开专门的网页，直接在 WhatsApp、Telegram、Discord 甚至微信（通过集成）给它发消息。这种感觉就像是在和一个极其靠谱的私人秘书沟通。
2. **它有真实的“手脚”：** 它可以直接操作你的电脑系统。读写文件、运行终端命令、操作浏览器去填表、抓取数据。比如你人在外面，突然想让家里的电脑编译一个项目，或者查询一下某份本地文档里的细节，发条信息，它就在后台默默帮你搞定了。
3. **它拥有“持久化记忆”：** 这是 Moltbot 最神奇的地方。一般的 AI 聊完就忘，但它能记住你的偏好、你的工作习惯、甚至是你们三天前聊过的某个灵感。随着你和它沟通越久，它就越像是一个懂你的“老伙计”。

### 它是如何改变工作流的？

在官方社区的反馈中，我们看到了许多令人拍案叫绝的玩法：

- **“带娃编程”：** 有个爸爸在哄孩子睡觉时，用手机在 Telegram 上给家里的 Moltbot 下指令，竟然从零开发并上线了一个简单的网站。
- **自动化管家：** 有人让它监控邮箱，自动处理那些繁琐的账单报销，甚至在机票开售时自动帮他完成登机校验。
- **AI 进化 AI：** 最硬核的玩法是，让 Moltbot 给自己写插件（Skills）。如果你觉得它缺个什么功能，直接告诉它，它会自己写代码、测试、然后给自己装上。

### 隐私与自由：你的数据你做主

很多人担心 AI 的隐私问题，Moltbot 给出的答案是：**彻底本地化**。

它是一个开源项目，所有的技能、上下文和运行逻辑都跑在你自己的机器上。你可以选择连接最强的 Claude 3.5 接口，也可以选择完全本地化的模型，这种安全性是任何闭源 SaaS 服务都无法比拟的。开发者 Peter Steinberger 甚至笑称，这让很多人为了跑它专门去买了一台 Mac Mini 放在家里当“AI 服务器”。

### 总结：不再是工具，而是队友

如果说 ChatGPT 是一个百科全书，那么 Moltbot 就是一个**数字队友**。

它最吸引人的地方在于那种“确定感”——你知道它就在那里，24 小时待命，能翻看你的笔记，能改你的代码，能帮你发邮件。你不再是单纯地“使用”AI，而是在“指挥”AI。

目前该项目正处于高速迭代期，从 Clawdbot 更名为 Moltbot 之后，安装和上手门槛也在不断降低。如果你有一台闲置的电脑，或者想体验一把真正的“贾维斯”式生活，这个项目绝对值得你折腾一下。

### 🔥使用命令

```markdown

# 安装
curl -fsSL https://clawd.bot/install.sh | bash 

# 启动
clawdbot gateway

# 停止
clawdbot gateway stop 

# 使用whatsapp登陆
clawdbot channels login --channel whatsapp

# 检查WhatsApp是否连通
curl -I https://web.whatsapp.com/

# 选择模型
clawdbot configure --section model  

# 1) 设置 MiniMax Anthropic-compatible Base URL（官方推荐）
clawdbot config set models.providers.minimax.baseUrl "https://api.minimaxi.com/anthropic"

clawdbot config set models.providers.minimax.baseUrl "https://api.minimax.io/anthropic"

# 2) 设置 MiniMax API Key（对应 MINIMAX_API_KEY）
clawdbot config set models.providers.minimax.apiKey "sk-cp-9CcEcNkvBGlY2IQnZt9p6a2dxE8P2pWf7Lys-T3mVu1rZ98IbayEdjErUw-9ufPXbVJVdxFK_mkIZzduCXcuCsSseXxh3xrlisnFepy77HNtzSZZDOE4y1k"

# 3)（建议）明确走 Anthropic Messages 兼容协议
clawdbot config set models.providers.minimax.api "anthropic-messages"

```
